## Status model + gates
```
Backlog ─(YOU approve)─▶ Approved ─(Claude)─▶ In Progress ─(Claude)─▶ In Review ─(YOU verify)─▶ Working
   └────────────────── Blocked ◀─(Claude, if stuck) ──────────────────┘
```
- **Approved** and **Working** are yours only (dev gate + done gate). Claude owns the middle.

## Transitions (every move stamps a dated comment)
```
python3 pgm/board.py new "title" [epic]  # scaffold next <KEY>-#####
python3 pgm/board.py approve <id>         # YOU:    -> Approved
python3 pgm/board.py start   <id>         # Claude: -> In Progress
python3 pgm/board.py review  <id>         # Claude: -> In Review
python3 pgm/board.py working <id>         # YOU:    -> Working
python3 pgm/board.py block   <id> "why"   # Claude: -> Blocked
python3 pgm/board.py                       # regenerate this board
```
